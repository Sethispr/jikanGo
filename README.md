# Jikan Go (IN DEVELOPMENT)

[![Go Report Card](https://goreportcard.com/badge/github.com/Sethispr/jikanGo)](https://goreportcard.com/report/github.com/Sethispr/jikanGo)

A modern Go wrapper which supports Jikan V1, V2, V3, V4.

**DISCLAIMER**: Not all features have full coverage, the only ones with full scopes are the anime, club, character, genre and user info searching.
